### *️⃣ Requirements

Windows:
1. Go to [Unity's download page](https://store.unity.com/download) and click “Download Installer for Windows”.
1. Install UnityHub.
1. With UnityHub opened click **ADD**
1. Select the project directory
1. Wait unity setup your project
